Układ strony

Układ strony, nazywany często layoutem, odpowiada za rozmieszczenie elementów na stronie internetowej. Dzięki CSS możemy decydować, czy elementy mają znajdować się obok siebie, jeden pod drugim, w siatce czy w konkretnym miejscu na ekranie. Właśnie za pomocą mechanizmów układu tworzy się menu, galerie zdjęć, sekcje stron oraz całe interfejsy aplikacji internetowych. W CSS istnieje kilka podstawowych sposobów kontrolowania układu strony. Najważniejsze z nich to właściwość display, system Flexbox, Grid oraz pozycjonowanie elementów.

display

Właściwość display określa sposób wyświetlania elementu na stronie. To od niej zależy, ile miejsca zajmuje element i jak zachowuje się względem innych elementów.

Najważniejsze wartości display:

  • block, element blokowy zajmuje całą dostępną szerokość swojego rodzica i zawsze rozpoczyna się od nowej linii, jeżeli na stronie znajdują się dwa lub więcej elementy blokowe, jeden zostanie umieszczony pod drugim
  • inline, element liniowy zajmuje tylko tyle miejsca, ile potrzebuje jego zawartość, nie rozpoczyna nowej linii, kilka elementów inline może znajdować się obok siebie w jednym wierszu.
  • inline-block, łączy cechy elementów blokowych i liniowych
  • flex (szczegóły poniżej)
  • grid (szczegóły poniżej)

Flexbox

Flexbox to nowoczesny system układania elementów w jednym wymiarze — poziomo lub pionowo. Jego głównym zadaniem jest ułatwienie rozmieszczania elementów oraz ich wyrównywania.Aby aktywować Flexbox, należy ustawić display: flex;. Domyślnie elementy układają się w jednym rzędzie. Dzięki Flexboxowi można bardzo łatwo:

  • układać elementy obok siebie
  • wyśrodkowywać elementy
  • tworzyć menu
  • budować responsywne układy

Aby idealnie wyśrodkować element należy użyć justify-content: center; które kontroluje rozmieszczenie elementów w poziomie i align-items: center; które kontroluje rozmieszczenie elementów w pionie.

Grid

CSS Grid to nowoczesny system tworzenia układów opartych na wierszach i kolumnach. Jest bardziej rozbudowany od Flexboxa i doskonale nadaje się do tworzenia całych layoutów stron. Aby aktywować Grid należy użyć display: grid; Aby stworzyć kolumny należy użyć grid-template-columns.

Jednostka fr oznacza część dostępnej przestrzeni w grid-template-columns, np. gdy użyjemy grid-template-columns: 2fr 1fr; to pierwsza kolumna będzie dwa razy szersza od drugiej.

Pozycjonowanie

Pozycjonowanie pozwala określić dokładne miejsce elementu na stronie. Domyślnie wszystkie elementy mają position:static;

Wartości position:

  • relative, element pozostaje w normalnym układzie strony, ale można go przesuwać względem jego pierwotnej pozycji
  • absolute, element zostaje wyjęty z normalnego układu strony i pozycjonowany względem najbliższego rodzica posiadającego pozycjonowanie
  • fixed, element zostaje przyklejony do okna przeglądarki, podczas przewijania strony pozostaje w tym samym miejscu
  • sticky, łączy cechy relative i fixed, element zachowuje się normalnie do momentu osiągnięcia określonej pozycji, a następnie „przykleja się” do ekranu
