Selektory

Selektory w CSS służą do wybierania elementów HTML, które chcemy ostylować. Dzięki nim przeglądarka wie, do których elementów mają zostać zastosowane konkretne reguły CSS. Można powiedzieć, że selektor to „wskazanie celu” dla stylu. Bez selektorów CSS nie wiedziałby, które elementy mają zmienić kolor, rozmiar czy położenie.

Selektor elementu

Selektor elementu jest najprostszym typem selektora. Wybiera wszystkie elementy danego rodzaju znajdujące się na stronie. Jeśli wybierzemy znacznik p, style zostaną zastosowane do każdego akapitu. Ten sposób jest bardzo wygodny, gdy chcemy nadać wspólny wygląd wszystkim elementom tego samego typu. Dzięki temu można szybko ustawić podstawowy wygląd strony, np. kolory tekstu, rozmiary nagłówków czy odstępy.

Selektor klasy

Selektor klasy jest jednym z najczęściej używanych selektorów w CSS. Pozwala przypisywać style tylko wybranym elementom, zamiast wszystkim elementom danego typu. Klasy są bardzo praktyczne, ponieważ można ich używać wielokrotnie na różnych elementach. Dzięki temu łatwo tworzyć powtarzalne style dla przycisków, sekcji, kart czy menu. W HTML klasa jest zapisywana za pomocą atrybutu class, a w CSS poprzedza się ją kropką.

Selektor ID

Selektor ID służy do wybierania jednego, konkretnego elementu na stronie. W przeciwieństwie do klas identyfikator powinien być unikalny, co oznacza, że nie powinno się używać tego samego ID więcej niż raz w jednym dokumencie. ID zapisuje się w HTML za pomocą atrybutu id, a w CSS poprzedza znakiem #.

Selektor grupowy

Selektor grupowy pozwala stylować kilka różnych elementów jednocześnie za pomocą jednej reguły CSS. Dzięki temu nie trzeba powtarzać tego samego kodu wiele razy. Elementy oddziela się przecinkami, a wszystkie otrzymują te same style.

Selektor uniwersalny

Selektor uniwersalny oznaczany jest gwiazdką * i wybiera wszystkie elementy na stronie. Jest on często używany do resetowania domyślnych stylów przeglądarki, takich jak marginesy czy paddingi. Dzięki temu można rozpocząć stylowanie od „czystej” strony.

Selektor zagnieżdżony

CSS pozwala wybierać elementy znajdujące się wewnątrz innych elementów. Dzięki temu można tworzyć bardziej precyzyjne style. Przykładowo można ostylować tylko akapity znajdujące się w określonej sekcji, bez wpływania na resztę strony.

Selektor dziecka

Selektor dziecka działa podobnie do selektora zagnieżdżonego, ale jest bardziej precyzyjny. Wybiera tylko bezpośrednie dzieci, a nie głębiej zagnieżdżone elementy.

Selektor sąsiada

Selektor sąsiada pozwala wybrać element znajdujący się bezpośrednio po innym elemencie.

Selektor pseudoklasy

Pseudoklasy opisują specjalny stan elementu. Pozwalają reagować na działania użytkownika lub pozycję elementu w dokumencie. Przeglądarka cały czas obserwuje elementy znajdujące się na stronie i sprawdza, w jakim są aktualnie stanie. Analizuje między innymi, czy kursor myszy znajduje się nad elementem, czy element został kliknięty, czy użytkownik wpisuje coś do formularza albo jakie miejsce zajmuje element w strukturze dokumentu. Gdy warunek określony przez pseudoklasę zostanie spełniony, przeglądarka automatycznie nakłada odpowiedni styl CSS. Nie trzeba uruchamiać żadnego dodatkowego kodu ani wykonywać zmian ręcznie, ponieważ cały mechanizm działa samodzielnie w tle. Najpopularniejszą pseudoklasą jest :hover, która działa po najechaniu myszką.

Istnieją tez inne pseudoklasy takie jak:

  • :focus
  • :active
  • :first-child
  • :last-child
  • :nth-child()
  • :visited
  • :link

Selektor pseudoelementu

Pseudoelementy pozwalają stylować konkretną część elementu, a nie cały element. Przykładowo można zmienić wygląd pierwszej litery lub pierwszej linii tekstu.

Przykłady użycia selektorów:

Selektor elementu:

p {
  color: blue;
}

Selektor klasy:

.box {
  background-color: gray;
}

Selektor ID

#title {
  color: red;
}

Selektor grupowy

h1, p, div {
  color: blue;
}

Selektor uniwersalny

* {
  margin: 0;
  padding: 0;
}

Selektor zagnieżdżony

div p {
  color: green;
}

Selektor dziecka

div > p {
  color: orange;
}

Selektor sąsiada

h1 + p {
  color: purple;
}

Selektor pseudoklasy

button:hover {
  background-color: red;
}

Selektor pseudoelementu

p::first-letter {
  font-size: 30px;
}
