Czcionki i tekst

Wygląd tekstu jest jednym z najważniejszych elementów każdej strony internetowej. To właśnie tekst przekazuje użytkownikowi informacje, dlatego powinien być czytelny, estetyczny i odpowiednio sformatowany. CSS pozwala kontrolować wiele aspektów związanych z wyglądem tekstu, takich jak rozmiar czcionki, jej krój, grubość, wyrównanie czy odstępy między znakami i liniami. Odpowiednie formatowanie tekstu wpływa nie tylko na wygląd strony, ale również na komfort czytania. Nawet najładniejsza strona może być niewygodna w odbiorze, jeśli tekst jest zbyt mały, źle wyrównany lub posiada niewłaściwe odstępy.

font-size

Właściwość font-size służy do określania rozmiaru tekstu. Dzięki niej można sprawić, że tekst będzie większy lub mniejszy. Najczęściej rozmiar podaje się w pikselach (px), ale można używać również innych jednostek, takich jak em, rem czy %. Im większa wartość, tym większy tekst zostanie wyświetlony na stronie.

font-family

Właściwość font-family określa krój czcionki używany przez element. Różne czcionki mogą całkowicie zmienić wygląd strony. Niektóre sprawiają wrażenie bardziej formalnych, inne nowoczesnych lub dekoracyjnych. Przeglądarka próbuje użyć pierwszej czcionki z listy. Jeśli nie jest ona dostępna na komputerze użytkownika, przechodzi do kolejnej.

font-weight

Właściwość font-weight odpowiada za grubość tekstu. Dzięki niej można wyróżniać ważne fragmenty treści bez konieczności używania dodatkowych znaczników HTML.

Najczęściej spotykane wartości font-weight to:

  • normal - zwykła grubość tekstu
  • bold - pogrubiony tekst
  • można również używać wartości liczbowych od 100 do 900

Istnieją również inne wartości font-weight takie jak:

  • lighter - cieńszy/lżejszy tekst
  • bolder - bardziej pogrubiony tekst

text-align

Właściwość text-align odpowiada za wyrównanie tekstu wewnątrz elementu. Dzięki niej można decydować, czy tekst ma znajdować się przy lewej krawędzi, prawej, na środku lub być wyjustowany. Wyjustowanie sprawia, że tekst jest równomiernie rozciągnięty pomiędzy lewym i prawym marginesem, podobnie jak w książkach czy gazetach.

Posiada wartości takie jak:

  • left
  • right
  • center
  • justify

Odstępy w tekście

CSS pozwala kontrolować różne rodzaje odstępów, co znacząco wpływa na czytelność treści. Jedną z najczęściej używanych właściwości jest line-height, która określa odstęp między kolejnymi liniami tekstu. Większa wartość powoduje zwiększenie odstępów między wierszami. Można również kontrolować odstępy między literami za pomocą letter-spacing. Inną właściwością jest word-spacing, która odpowiada za odstępy między wyrazami. Zbyt duże odstępy mogą utrudniać czytanie, dlatego należy używać ich z umiarem. Odpowiednio dobrane odstępy sprawiają, że tekst wygląda bardziej profesjonalnie i jest łatwiejszy do przyswojenia przez użytkownika.

Przykłady użycia właściwości:

p {
    font-size: 26px;
}

Wielkość liter 26px

body {
    font-family: Arial, sans-serif;
}

Tekst napisany czcionką sans-serif

p {
    font-weight: normal;
}

Tekst napisany zwykłą grubością

p {
    font-weight: bold;
}

Tekst pogrubiony

p {
    font-weight: 200;
}

Tekst napisany z grubością 200

p {
    text-align: left;
}

Tekst ustawiony po lewej stronie

p {
    text-align: right;
}

Tekst ustawiony po prawej stronie

p {
    text-align: center;
}

Tekst ustawiony na środku

p {
    text-align: justify;
}
