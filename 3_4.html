Kolory i tła

Kolory i tła są bardzo ważnym elementem tworzenia stron internetowych. To właśnie dzięki nim strony wyglądają estetycznie, nowocześnie i są przyjemniejsze w odbiorze dla użytkownika. Kolory pomagają wyróżniać elementy, zwracać uwagę użytkownika na ważne informacje oraz budować odpowiedni wygląd i klimat strony. Tła natomiast pozwalają organizować wygląd strony, oddzielać od siebie sekcje i tworzyć bardziej rozbudowane projekty.

Kolory

W CSS kolory wykorzystuje się przede wszystkim do zmiany wyglądu tekstu, przycisków, obramowań oraz wielu innych elementów strony. Najczęściej używa się do tego właściwości color, która odpowiada za kolor tekstu.

CSS pozwala zapisywać kolory na kilka różnych sposobów:

  • Nazwy kolorów, jest to najprostszy sposób zapisu kolorów, CSS posiada wiele gotowych nazw, takich jak red, blue, green czy black
  • HEX, kolor zapisuje się za pomocą znaku # oraz sześciu znaków określających ilość czerwonego, zielonego i niebieskiego, zapis jest oparty na systemie szesnastkowym, który opisuje kolor przez trzy wartości: czerwony, zielony i niebieski, każda z nich ma zakres od 00 do ff
  • RGB, które oznacza Red, Green, Blue, każdy z tych kolorów posiada wartość od 0 do 255
  • RGBA, działa jak RGB, ale posiada dodatkową wartość odpowiadającą za przezroczystość, dzięki temu można tworzyć półprzezroczyste elementy
  • HSL, który składa się z trzech wartości: Odcień (Hue) to podstawowy kolor na kole barw. Jest podawany w stopniach od 0 do 360. Każda wartość odpowiada innemu kolorowi, na przykład czerwony, zielony, niebieski i wszystkie ich przejścia. Można to porównać do obracania się po kole kolorów. Nasycenie (Saturation) określa intensywność koloru. Im wyższa wartość, tym kolor jest bardziej żywy i „czysty”. Gdy nasycenie jest niskie, kolor staje się szary i mniej wyraźny. Jasność (Lightness) określa, jak jasny lub ciemny jest kolor. Przy niskich wartościach kolor jest ciemny, przy wysokich bardzo jasny, a przy wartości 50% jest najbardziej „normalny”.
  • HSLA, działa jak HSL, ale posiada dodatkową wartość odpowiadającą za przezroczystość, dzięki temu można tworzyć półprzezroczyste elementy

Tła

Tła w CSS odpowiadają za wygląd przestrzeni znajdującej się za elementami strony. Mogą być jednokolorowe, zawierać obrazy lub wykorzystywać gradienty. Dzięki tłom strony internetowe wyglądają bardziej nowocześnie i estetycznie.

Właściwości tła:

  • background, które pozwala ustawić wiele parametrów tła jednocześnie w jednym zapisie, zamiast pisać kilka osobnych właściwości, można wszystko połączyć w jedną linię
  • background-color, które pozwala ustawić jednolity kolor tła elementu
  • background-image, które pozwala ustawić obraz jako tło elementu
  • background-position, które określa, gdzie dokładnie ma się znajdować tło
  • background-size, określa rozmiar tła, używa się go głównie przy obrazach, posiada kilka wartości takich jak:
    • cover, obraz wypełnia caly element
    • contain, obraz mieści się cały w elemencie
    • konkretne wartości, np. 100px
  • background-repeat, które decyduje czy tło ma się powtarzać
  • background-origin, które określa od jakiego miejsca ma się zaczynać rysowanie tła
  • background-clip, które decyduje, gdzie tło ma się kończyć
  • backgroud-attachment, które określa, czy tło ma się poruszać razem ze stroną

Przykłady użycia kolorów i tła:

Nazwy kolorów:

p {
  color: red;
}

Kolor czerwony

HEX:

p {
  color: #00ff00;
}

Kolor zielony

RGB:

p {
  color: rgb(0, 0, 255);
}

Kolor niebieski

RGBA:

p {
  color: rgba(0, 0, 255, 0.5);
}

Kolor niebieski częściowo przeźroczysty

HSL:

p1 {
  color: hsl(0, 100%, 50%);
}

Kolor żółty

HSLA:

p1 {
  color: hsla(0, 100%, 50%, 0.5);
}

Kolor żółty częściowo przeźroczysty

background

div {
  background: #3498db url("tlo.jpg") center/cover no-repeat fixed;
}

background-color

div {
  background-color: lightgray;
}

background-image

div {
  background-image: url("tlo.jpg");
}

background-position

div {
  background-position: center;
}

background-size

div {
  background-size: cover;
}

background-repeat

div {
  background-repeat: no-repeat;
}

background-origin

div {
  background-origin: padding-box;
}

background-clip

div {
  background-clip: border-box;
}

backgroud-attachment

div {
  background-attachment: scroll;
}
